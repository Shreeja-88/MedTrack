## 📌 Project Overview

MedTrack is a healthcare inventory and patient management system designed for:

- Small Clinics
- Pharmacies
- Medical Stores
- Academic Projects

It allows users to:

- Manage medicines
- Track expiry dates
- Monitor stock levels
- Register patients
- Assign medicines to patients
- Apply discounts & calculate final price
- View dashboard statistics

This version uses **local JSON storage**, making it simple to run without database setup.

# 🔌 API Endpoints

## Medicines

| Method | Endpoint              | Description       |
| ------ | --------------------- | ----------------- |
| POST   | /api/medicines        | Add medicine      |
| GET    | /api/medicines        | Get all medicines |
| PUT    | /api/medicines/:id    | Update medicine   |
| DELETE | /api/medicines/:id    | Delete medicine   |
| POST   | /api/reduce-stock/:id | Reduce stock      |

---

## Patients

| Method | Endpoint          | Description      |
| ------ | ----------------- | ---------------- |
| POST   | /api/patients     | Add patient      |
| GET    | /api/patients     | Get all patients |
| PUT    | /api/patients/:id | Update patient   |
| DELETE | /api/patients/:id | Delete patient   |

---

## Assignment

| Method | Endpoint                                      | Description       |
| ------ | --------------------------------------------- | ----------------- |
| POST   | /api/assign-medicine                          | Assign medicine   |
| DELETE | /api/assign-medicine/:patientId/:assignmentId | Remove assignment |
